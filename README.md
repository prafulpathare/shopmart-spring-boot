# Shopmart APIs

#### Procedure
1. Start MySQL server
2. Start Application
```php
  $ mvn clean install compile package
  $ java -jar ./taget/baseApi-0.0.1-SNAPSHOT.jar
```

<br />

#### APIs and Functionalities
- User
  - Address
- Customer *as User*
  - Order
- Supplier *as User*
  - Product
- Admin *as User*
